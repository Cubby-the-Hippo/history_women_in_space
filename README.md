# Breaking the Sky: Women in Aviation and Space Exploration

An interactive virtual museum exhibit and choose-your-own-adventure educational experience built with React, Tailwind CSS, and Framer Motion.

## Features

- Cinematic landing page with animated starfield
- Museum-style introduction with historical context
- Interactive timeline (1921–1995)
- Virtual exhibit hall with six featured pioneers
- Branching narrative adventure (4 historical paths)
- Artifact gallery with unlock system
- Modern connections to women in STEM today
- Professional bibliography
- Reflective conclusion

## Getting Started

```bash
npm install
npm run dev
```

Open the URL shown in the terminal (typically http://localhost:5173).

## Build for Production

```bash
npm run build
npm run preview
```

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS v4
- Framer Motion
